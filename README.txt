ETS 토익 기출 보카 PWA v2

변경점
- 데이터 분리(JSON fetch)로 초기 렌더링 부담 완화
- 라이트/다크/자동 테마 지원
- 데스크톱 UI 개선(테이블형 단어장, 더 정돈된 탭/통계/필터)
- service worker 캐시 갱신

배포 방법
1. 이 폴더 안의 파일을 GitHub 저장소 루트에 업로드
2. Settings > Pages > Deploy from a branch > main / root
3. 배포 주소 접속 후 홈 화면에 추가

주의
- 브라우저 저장(localStorage) 방식이라 오답/암기 기록은 기기별로 따로 저장됩니다.
- 기존 v1 저장 기록은 v2와 키가 달라 자동 이전되지 않습니다.
